# 02 — 信息架构与界面交互

> 描述当前已实现的主界面结构（随产品演进更新）。

## 1. 设计目标

一屏完成主任务：**发现技能 → 看懂技能 → 选中目标项目 → 复制 / 应用 Bundle**。  
轻量：无仪表盘堆砌；首屏即工作台。

## 2. 导航结构

```
App
├── 技能库（默认）          # 两栏工作台 + 详情二级窗
├── 组合包                  # 常用技能组合
├── 健康检查                # 健康报告列表
├── 新建项目                # 目录脚手架 + 选技能 + 在线安装 + 健康检查
├── 在线安装                # skills.sh / npx skills
├── 设置                    # 策略模板、ZIP 导入等
├── 来源与项目              # 扫描源开关、项目根登记
└── 操作记录                # OpLog
```

以上入口均已可用。

## 3. 技能库布局

```text
┌──────────────┬──────────────────────────────────────────────┐
│ Left Rail    │ Catalog                                      │
│ ~232px       │                                              │
├──────────────┼──────────────────────────────────────────────┤
│ 搜索 (/)     │ 工具条：计数 / 多选 / 复制 / ZIP …           │
│ Runtime 筛选 │ 目标项目条（路径 · runtime · 冲突 · 拖放）   │
│ 副本 / 收藏  │ 操作预览（有复制预览时展开）                 │
│ 标签筛选     │ Skill 卡片网格                               │
│ 常用入口     │                                              │
│ Bundle 短列表│                                              │
└──────────────┴──────────────────────────────────────────────┘

点击卡片 → 右侧滑出二级详情窗（独立滚动；Esc / 点遮罩关闭）
```

### 3.1 左栏

| 区块 | 内容 | 交互 |
|------|------|------|
| Search | name / description | `/` 聚焦；过滤中栏 |
| Runtime | cursor / claude / agents / … | 多选 AND |
| 开关 | 仅看有副本、仅看收藏 | |
| 标签 | 用户标签列表 | 单选筛选；详情内增删 |
| 常用 | 最近使用 / 收藏快捷入口 | 点选打开详情窗 |
| Bundles | 短列表 | 进入组合包页 |

### 3.2 中栏（Catalog）

**顶部目标项目条：**

| 元素 | 说明 |
|------|------|
| 当前项目 | 路径；未选择时提示 |
| 选择项目 / 就绪向导 | 浏览文件夹；分析技术栈 |
| 写入 runtime | agents / claude / cursor 多选 |
| 冲突策略 | 覆盖 / 跳过 / 重命名 |
| 拖放区 | 卡片拖入生成复制预览；预览条可执行 / 取消 |

**卡片字段（列表不加载全文）：**

- `name`、截断 `description`
- 徽章：`runtime`、`scope`、`RO`、`scripts`、`twins`、标签、健康分

**交互：**

| 操作 | 行为 |
|------|------|
| 单击 | 选中并打开详情二级窗 |
| 多选模式 / Ctrl/⌘ / Shift | 勾选，不打开详情 |
| 拖到目标项目条 | 生成预览 → 确认执行 |
| 工具条 | 预览复制、导出 ZIP、收藏、删除 |

### 3.3 详情二级窗

从右侧滑出；标题栏固定，正文区独立滚动。内容包括：

1. **标题区**：name、路径、徽章、打开目录 / 提取副本 / 源码切换 / 关闭  
2. **导语**：description 全文  
3. **标签**：增删（扫描时保留）  
4. **健康**：有问题时展示摘要；完整报告见「健康」页  
5. **副本**：一致 / 有差异；查看 diff；以当前为准同步  
6. **脚本风险 / 变更历史**：有则显示；历史默认折叠  
7. **附属文件**：chip 列表  
8. **正文**：大纲可折叠；友好 Markdown 或源码视图  

不做内嵌可视化编辑器；「打开目录」用资源管理器处理文件。

## 4. 关键用户流程

### F1 — 搞清本机有哪些 skill

1. 启动加载索引；可点「重新扫描」  
2. 左栏按 Runtime / 副本 / 标签过滤  
3. 点开卡片看详情  

### F2 — 新建项目复用组合

1. 技能库顶部选目标项目  
2. 组合包页选 Bundle → 应用到目标项目  
3. 或向导页按技术栈生成 Bundle 后应用  
4. 日志页可追溯  

### F3 — 从只读插件提取自用

1. 打开 RO skill 详情  
2. 「提取副本」→ 写入 `~\.agents\skills`  

### F4 — 清理重复副本

1. 勾选「仅看有副本」  
2. 详情中 Diff → 同步或删除多余（仅 RW）  

## 5. 其它页面（摘要）

| 页 | 能力 |
|----|------|
| 组合包 | 列表、创建、导入导出 JSON、应用到项目、删除 |
| 健康 | 报告列表、展开问题、打开目录、重新检查；skills.sh 对照 diff |
| 向导 | 分析目标项目栈、推荐 skill / Bundle |
| Registry | `find` / `list` / `add` / `update` / `remove` |
| 设置 | 策略模板、ZIP 导入、插件拷贝限制 |
| 源与项目 | 启用源、登记项目根 |
| 日志 | 近期写操作 |

## 6. 交互与安全

| 规则 | 说明 |
|------|------|
| 拖拽默认复制 | 生成预览后执行 |
| 删除二次确认 | 展示副本数、Bundle 引用；进回收站 |
| RO 源 | 禁止就地删除/覆盖；可提取副本 |
| 键盘 | `/` 搜索；`Del` 删除（确认）；`Esc` 关提示 / diff |

## 7. 视觉与轻量约束

- 单窗口；无强制托盘常驻  
- 详情按需读盘；列表只绑索引  
- 不内嵌完整 IDE / Agent 对话
