# 使用指南

面向最终用户的简要操作说明。开发者请先阅读仓库 [README](../README.md)。

## 1. 第一次打开

1. 安装并启动 SSM（Windows x64）
2. 点击顶部 **重新扫描**，等待索引本机技能源
3. 若项目级技能看不到：打开 **来源与项目**，登记项目根目录后再扫描
4. 若要用 skills.sh 安装：本机需已安装 Node.js

## 2. 浏览与搜索

- 中间为技能卡片；左侧可按适用工具、副本、收藏、标签筛选
- 按 `/` 聚焦搜索框（匹配名称 / 说明）
- 单击卡片：弹出详情二级窗（独立滚动；`Esc` 关闭）
- 界面上的 **?** 可悬停查看简短说明
- 徽章含义：只读 · 有脚本 · 副本 · 健康分等

## 3. 复制到项目

1. 在技能库顶部 **目标项目** 条中 **选择项目**（或先在「来源与项目」登记）
2. 勾选要复制到哪些工具（Cursor / Claude / Agents）
3. 将卡片 **拖入** 目标项目条，或工具条 **复制到项目**
4. 核对预览路径后点 **确认复制**

「文件已存在时」的处理方式可在目标项目条或 **设置** 中调整。

## 4. 副本与同步

同名技能出现在多处时，卡片会标「副本」：

1. 打开详情 → **副本**
2. **diff** 查看 `SKILL.md` 差异
3. 确认后可用 **同步**（以当前为准覆盖另一侧；只读目标不可覆盖）

## 5. 新建项目（推荐）

在 **新建项目** 页可一次完成新项目技能目录搭建：

| 步骤 | 做什么 |
|------|--------|
| 1. 项目目录 | 选择文件夹；可 **分析技术栈并推荐**，把匹配技能加入已选 |
| 2. 工具目录 | 勾选 `.claude/skills`、`.agents/skills`、`.cursor/skills`（默认全选）；设定冲突策略 |
| 3. 本机技能 | 搜索并多选要复制进去的技能 |
| 4. skills.sh | 打开**系统终端**（工作目录=本项目），执行基础 `npx skills …`，选项在终端里选 |
| 5. 一键落地 | 创建勾选的空目录 → 复制已选本机技能 → 对本项目做健康检查 |
| 6. 健康结果 | 列出项目内技能；标出可能非最新（与远端不一致或副本偏旧） |

终端装完后请再点「重新扫描」或本页「重新检查本项目」。

## 6. 组合包

1. 在技能库多选技能，左侧填写名称创建；或在 **组合包** 页管理  
2. 选好目标项目后 **应用到目标项目**  
3. 支持组合包 JSON 导入 / 导出，便于分享  

## 7. 健康检查

1. 打开 **健康检查** 页，可 **重新检查全部**
2. 展开卡片查看问题；可 **打开目录** 到资源管理器
3. 部分问题（如缺少 name）支持 **应用修复**
4. 仅通过 `npx skills` 安装、且写入锁文件的技能，才会对照 skills.sh 远端版本并显示 diff

本地手写的技能 **不会**因为「不在锁文件里」被扣分。

## 8. 在线安装（skills.sh）

搜索 / 安装 / 更新 / 移除会 **打开系统终端**（优先 Windows Terminal，否则 `cmd`），并自动执行最基础的命令，例如：

```text
npx skills find [关键词]
npx skills add owner/repo -g          # 「在线安装」页：全局
npx skills add owner/repo             # 「新建项目」页：在项目目录下
npx skills update [-g]
npx skills remove [名称] [--global]
```

**不会**预填 `--yes`、`--copy`、`-a`、`-s` 等；symlink / copy、选哪些技能、装到哪些工具，都在终端交互里由你决定。

| 入口 | 范围 |
|------|------|
| **新建项目** | 终端 `cwd` = 当前项目（项目级） |
| **在线安装** | 全局（安装命令带 `-g`） |

「已安装列表」仍在应用内直接显示输出。  
需本机 Node.js；终端操作完成后请点「重新扫描」。

## 9. 导入导出与其它

| 能力 | 入口 |
|------|------|
| 选中技能打 ZIP | 技能库工具条 |
| 导入 ZIP | 设置页 |
| 只读源提取自用 | 详情「提取为自己的副本」→ 通常写入 `~\.agents\skills` |
| 删除 | 工具条或 `Del`（确认后进回收站） |
| 操作追溯 | **操作记录** 页 |

## 10. 快捷键

| 键 | 作用 |
|----|------|
| `/` | 搜索 |
| `Del` | 删除选中（仍确认） |
| `Esc` | 依次关闭：副本 diff → 详情窗 → 状态条 |
